# X線被ばく線量推計システム v2.0

患者被ばく線量（臓器線量・実効線量）を推計するWebアプリです。  
Japan DRL 2025（日本の診断参考レベル 2025年版）に対応しています。

## 対応モード

| モード | 内容 |
|--------|------|
| 📷 一般X線撮影 | 胸部・腹部・脊椎・頭部・四肢など 17部位 |
| 🔄 CT検査 | 成人CT 10プロトコール + 小児CT年齢別DRL自動対応 |
| 🔬 診断透視 | 9種別（据置形）+ 胃X線検診2種別 |
| 🩻 マンモグラフィ | 2D・DBT・PMMA 40mm |
| 🦷 パノラマ撮影 | PKA・DWP vs DRL 2025 |

## 使い方

### ブラウザで直接開く（推奨・インターネット不要）

```
xray-dose-calculator.html をダブルクリック
```

Chart.js はファイルに組み込み済みのため、**インターネット接続不要**で動作します。

### ローカルサーバーで起動する場合

```bash
npx http-server . -p 8080
# → http://127.0.0.1:8080/xray-dose-calculator.html
```

## 参照規格・出典

- **Japan DRL 2025** — 日本の診断参考レベル（2025年版）
  - 表3.1.1 成人CT / 表3.1.2 小児CT
  - 表3.2 一般撮影 / 表3.3 マンモグラフィ
  - 表3.4.2 パノラマ撮影
  - 表3.6.1 透視（据置形）/ 表3.6.3 胃X線検診
- **ICRP Publication 103** (2007) — 組織荷重係数
- **NRPB-SR262** (1994) — X線規格化臓器線量係数
- **Shrimpton et al. 2006** — CT k係数（DLP法）

## 注意事項

本ツールは公表された変換係数を用いた**簡易推計ツール**です。  
モンテカルロ法による個人化計算（PCXMC等）とは異なります。
